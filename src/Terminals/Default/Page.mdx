<details>
	<summary>Installation</summary>
	
    1. Get the [model]("https://google.com")
    2. Place the model in `Terminal > Addons`
    3. Set the terminal part and edit the default config values
    4. Load the terminal in the server file using `wrapper:LoadTerminal(script.Parent.Addons.Default)` **before loading any addon**
</details>
