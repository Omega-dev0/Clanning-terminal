Multipoint domination terminal

<details>
	<summary>Installation</summary>
	
    1. Get the [model]("https://google.com")
    2. Place the model in `Terminal > Addons`
    3. Add/remove object values in `Configuration > Terminals` to add terminals
    4. Load the terminal in the server file using `wrapper:LoadTerminal(script.Parent.Addons.Domination)` **before loading any addon**
</details>
