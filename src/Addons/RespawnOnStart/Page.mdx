# Respawn on start

:::note[Metadata]
	Name: Respawn on start <br/>
	Description: Respawn players in playing teams on terminal start <br/>
	Version: v1.1 <br/>
	Author: Omega77073 <br/>
	Compatibility: >=1.2.0 <br/>
:::


Respawns attackers and defenders on terminal start.

<details>
	<summary>Installation</summary>
	
    1. Get the [model]("https://create.roblox.com/store/asset/91476807453681")
    2. Place the model in `Terminal > Addons`
    3. Load the addon in the server file using `wrapper:AddAddon(script.Parent.Addons.RespawnOnStart)`
</details>
