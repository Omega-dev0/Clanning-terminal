# Cloud logging addon

:::note[Metadata]
	Name: Cloud Logging <br/>
	Description: The default cloud logging addon <br/>
	Version: v1.1 <br/>
	Author: Omega77073 <br/> 
	Compatibility: >=1.2.0 <br/>
:::



This module provides free logging of events and commands used during that events in a [website](https://terminal.omegadev.xyz), perfect to check the scoreboard or for admin abuse.


<details>
	<summary>Installation</summary>
	
    1. Get the [model]("https://google.com")
    2. Place the model in `Terminal > Addons`
    3. Edit the configuration instance's attributes to add your api key and private key
    4. Load the addon in the server file using `wrapper:AddAddon(script.Parent.Addons.CloudLogging)`
</details>

:::warning[API KEY]
    This addon requires an API key, to get one for free DM `@omega7707`, HttpService should also be enabled !
:::
