# UI boilerplate

:::note[Metadata]
	Name: Terminal UI Boilerplate <br/>
	Description: A boilerplate for creating terminal UI addons <br/>
	Version: v1.1 <br/>
	Author: ------- <br/>
	Compatibility: >=1.2.0 <br/>
:::


To integrate your UI, simply place your UI in the screen gui and edit the controller script

<details>
	<summary>Installation</summary>
	
    1. Get the [model]("https://create.roblox.com/store/asset/140022108861968")
    2. Place the model in `Terminal > Addons`
    3. Load the addon in the server file using `wrapper:AddAddon(script.Parent.Addons.UI)`
</details>
