# Reset leaderboard on start

:::note[Metadata]
	Name: Reset leaderboard on start <br/>
	Description: Reset players' leaderboard stats on terminal start <br/>
	Version: v1.1 <br/>
	Author: Omega77073 <br/>
	Compatibility: >=1.2.0 <br/>
:::

Reset players' leaderboard stats on terminal start.

All `NumberValue` and `IntValue` instances in the `leaderstats` folder will be reset to 0.

<details>
	<summary>Installation</summary>
	
    1. Get the [model]("https://create.roblox.com/store/asset/111610013059902")
    2. Place the model in `Terminal > Addons`
    3. Load the addon in the server file using `wrapper:AddAddon(script.Parent.Addons.ResetLeaderboardOnStart)`
</details>
